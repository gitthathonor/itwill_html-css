position 요소의 위치 지정법 지정

  • position:static; (기본값) 고정.
  • position:relative; 현재 위치에서 ~ 로 이동
  • position:absolute; 상위요소를 기준으로 ~ 이동
  • position:fixed; 브라우저를 기준으로 ~에 고정
  • position:sticky; 특정위치에서 부터 ~에 고정

z-index:n; 겹친 요소의 위아래값 변경

z-index값은 태그 추가시 1씩 자동 증가함. 요소들이 겹칠 때 상대적으로 해당 값이 클수록 위로 올라옴. 변경을 원하면 값을 조정(음수도 가능)

position:sticky; 특정위치에서 부터 ~에 고정

position:sticky;
relative와 fixed의 믹스. (적용대상은 body이외의 상위요소가 없어야 함)
