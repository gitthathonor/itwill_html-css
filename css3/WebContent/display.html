display - 요소가 화면에 나타나는 방식 지정

display의 값들

  • display:inline; block인 요소를 inline으로 바꿈
  • display:block; inline인 요소를 block으로 바꿈
  • display:inline-block; inline과 block을 합친 성격으로 바꿈
  • display:none; 화면에서 숨김

box-model의 특징

  1. inline 요소 (자동 줄 안바뀜)
    - 크기 적용 불가. 상하 margin적용 불가. 한 줄에 나란히 배치.
  2. block 요소(자동 줄바뀜)
    - 크기 적용 가능. 상하좌우 margin적용 가능. 한 줄에 나란히 배치 불가.
