html5에서 추가된 input type의 값들

input type="time" 날짜나 시간 (오전/오후 시 분)

시분

input type="date" 날짜나 시간 (연월일)

연월일

input type="datetime-local" 날짜나 시간 (연월일 오전/오후 시 분)

연월일시분

input type="week" 날짜나 시간 (연주)

연주

input type="month" 날짜나 시간 (연월)

연월

input type="number" 숫자

숫자형식의 데이터 (증감버튼 생성)
min="0" 로 최소값 지정
max="100" 로 최대값 지정
step="5" 로 증감간격 지정

input type="range" 범위내의 값

input type="tel" 연락처 (전화번호)

전화번호 형식의 데이터(모바일브라우저에서 가상키패드 변동, 전화걸기 가능)

input type="email" 연락처 (email주소)

email형식의 데이터(브라우저에서 email형식이 맞는지 체크해줌 @유무 등)

input type="url" 연락처 (웹주소,url주소)

웹주소(url)형식의 데이터(브라우저에서 url형식이 맞는지 체크해줌. 프로토콜 유무)

input type="color" 컬러값


input type="search" 검색용 상자

input type="file" 파일 전송용
